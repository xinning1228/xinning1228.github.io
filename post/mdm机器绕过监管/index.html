MDM机器绕过监管

6.1重新安装系统

  1. 电脑关机状态下，键盘长按command+R键不放，同时按一下开机键松手，此时继续长按command+R键不放，直到出现苹果图标。
  2. 选择“磁盘工具”，然后“继续”
  3. 选择系统磁盘，点击抹掉。这里可以看到磁盘使用情况，以及剩余情况。
  4. 弹出提示要抹掉“mac os”吗？这里一定要注意磁盘格式一定要选择APFS格式.
  5. 抹点进程完成，点击完成
  6. 回到macOS实用工具界面，选择重新安装 macOS，点击继续。
  7. 初始安装将运行大约 1 小时，并重新启动一次
  8. 然后它将显示大约 10-15 分钟的剩余时间
  9. 当它再次重新启动时，请务必按command-R启动进入恢复并继续执行主要程序

6.2主要程序

  1. 打开终端输入以下代码
$ csrutil disable
$ reboot
  1. 再次进入恢复模式
  2. 进入磁盘工具,安装HD盘
  3. 退出磁盘工具,打开终端输入以下代码
$ cd "/Volumes/Macintosh HD/System/Library"
$ cd ../../etc
$ echo "0.0.0.0 iprofiles.apple.com" >> hosts
$ echo "0.0.0.0 mdmenrollment.apple.com" >> hosts
$ echo "0.0.0.0 deviceenrollment.apple.com" >> hosts
$ echo "0.0.0.0 gdmf.apple.com" >> hosts
$ csrutil enable
$ reboot
  1. 如果您来到“选择您的国家/地区”对话框，请确保不要选择无线网络，而是“在没有互联网连接的情况下继续”
  2. 正常启动后，您可以在终端中验证 DEP 状态：
$ profiles status -type enrollment
Enrolled via DEP: No
MDM enrollment: No
